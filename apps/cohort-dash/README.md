# cohort Dash

## Tables

List of Tables with their function described below:

### cohort

Each record represents one cohort (aka site-wide group).

#### Fields

- id
- name
- component
- contextid
- description
- descriptionformat
- idnumber
- theme
- visible
- created_at
- updated_at

### cohort_members

Link a user to a cohort.

#### Fields

- id
- cohort_id
- created_at
- updated_at
- user_id
